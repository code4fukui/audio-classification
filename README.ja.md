# 音声識別

このプロジェクトは、MediaPipeライブラリを使用したウェブベースの音声識別デモを提供します。

## 機能
- MediaPipeのYAMNetモデルを使用したリアルタイム音声識別
- 話し声、笑い声、泣き声など、さまざまなカテゴリに音声を分類
- 識別されたカテゴリと信頼度スコアを表示

## 要件
このプロジェクトでは、WebAudio APIをサポートするモダンウェブブラウザが必要です。

## 使い方
1. ウェブブラウザでデモページを開きます。
2. 「音声識別開始」ボタンをクリックして、マイクからの音声識別を開始します。
3. 識別されたカテゴリと信頼度スコアが結果エリアに表示されます。

## データ / API
このプロジェクトでは、MediaPipeライブラリが提供するYAMNet音声識別モデルを使用しています。
